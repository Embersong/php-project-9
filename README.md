# Анализатор страниц

Test project "Анализатор страниц" for Hexlet "php" course

## Setup
Setup SSH before clone:

git clone https://github.com/Embersong/php-project-9
cd php-project-9

make install

## Launch
    make start

## Prerequisites
    Linux, Macos, WSL
    PHP >=8.2
    Make
    Git
