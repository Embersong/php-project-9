# Анализатор страниц

Test project "Анализатор страниц" for Hexlet "php" course

## Setup
git clone https://github.com/Embersong/php-project-9

cd php-project-9

make install

## Launch
    make start

## Prerequisites
    Linux, Macos, WSL
    PHP >= 8.3
    Make
    Git
